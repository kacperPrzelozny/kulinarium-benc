# 🥘 Kulinarium - Twój Osobisty Przewodnik Kulinarny

![Kulinarium - Food Guide](https://images.unsplash.com/photo-1504674900247-0877df9cc836?q=80&w=1200&auto=format&fit=crop)

## 📝 Opis aplikacji
**Kulinarium** to nowoczesna aplikacja typu **PWA (Progressive Web App)**, która służy jako cyfrowy dziennik Twoich kulinarnych odkryć. Została stworzona z myślą o smakoszach, którzy chcą dokumentować odwiedzone restauracje, ocenione dania oraz zachowywać dokładne lokalizacje GPS ulubionych miejsc.

### Kluczowe funkcjonalności:
* **Działanie Offline:** Dzięki technologii Service Worker aplikacja działa nawet bez dostępu do Internetu.
* **Zapis Multimediów:** Możliwość robienia zdjęć dań bezpośrednio z aparatu z automatyczną kompresją obrazu.
* **Geolokalizacja:** Precyzyjne zapisywanie współrzędnych GPS miejsca, w którym jadłeś.
* **Udostępnianie:** Integracja z systemowym panelem udostępniania (Web Share API).
* **Bezpieczeństwo danych:** Możliwość eksportu i importu całej bazy danych do pliku JSON.
* **Interaktywny UI:** Nowoczesny interfejs zbudowany na Bootstrap 5 z systemem powiadomień (Toasts).

### Technologie:
* **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5.
* **Baza danych:** IndexedDB (przechowywanie lokalne w przeglądarce).
* **PWA:** Web App Manifest, Service Workers.

## 🚀 Jak uruchomić projekt?
1. Sklonuj repozytorium.
2. Uruchom plik `index.html` korzystając z lokalnego serwera (np. rozszerzenie **Live Server** w VS Code).
3. Aby zainstalować aplikację na telefonie, wejdź na stronę przez HTTPS i wybierz opcję "Dodaj do ekranu głównego".
